# Customer Support Ticket Analyzer
## Project Overview
The Customer Support Ticket Analyzer was developed to store, clean, analyze, and extract insights from customer service tickets. The system uses Python dictionaries, lists, loops, functions, and string processing techniques to generate meaningful insights from customer issue descriptions.
## Objectives
-	To organize customer ticket data using Python data structures
-	To clean and standardize textual issue descriptions
-	To analyze keyword frequency within customer complaints
-	To evaluate ticket priority distribution
-	To identify the ticket containing the longest issue description
-	To extract unique words from customer feedback

## Data Overview:
-	Initial Tickets Loaded: 10
-	Additional Tickets Added: 3
-	Total Tickets Analyzed: 13
The ticket data was stored using a dictionary of lists.
Each ticket contains the following attributes:
-	Ticket Number
-	Customer Name
-	Issue Description
-	Priority Level (High / Medium / Low)
This structure ensures that all ticket-related information is connected through indexing.
## Data Cleaning Process
To improve analysis accuracy, the issue descriptions were processed using the following steps:
-	Converted all text to lowercase
-	Removed punctuation marks (.,!?-)
-	Replaced informal words (e.g., “ok” → “okay”)
-	Removed extra spaces
-	Normalized spacing between words
This ensured uniform formatting and improved keyword matching accuracy.
##  Key Findings
1.	Response time and service quality are recurring themes.
2.	Both positive and negative feedback are present.
3.	High-priority tickets require structured management.
4.	Text cleaning significantly improves analytical accuracy.
## Conclusion
The Python-based Ticket Analysis System successfully demonstrates:
-	Data Loading & Structuring
-	Text Cleaning & Processing
-	Keyword Analysis
-	Priority Classification
-	Maximum Value Detection
-	Unique Word Extraction
This project shows how basic Python tools can be applied to perform meaningful customer feedback analysis and generate actionable insights.
